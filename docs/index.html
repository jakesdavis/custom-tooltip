Custom Tooltip Plugin

Custom Tooltip plugin is able to create dynamic tooltips for a container.

Version:

1.0

Author:

jakesdavis (Jakes Davis)

Dependency:

jquery-2.1.1.js

Usage

$("#elementID").customTooltip();
$("#elementID").customTooltip({
	"position": "n"
});

Parameters

Name Type Description
border String Set border width of tooltip-box
position String Default position of tooltip wrt container
events - hover Boolean Set trigger for displaying the tooltip
events - click Boolean Set trigger for displaying the tooltip
events - focus Boolean Set trigger for displaying the tooltip
animate - effect - show String effect to show on display of tooltip
animate - effect - hide String effect to show on hide of tooltip
animate - duration Integer Duration of animation effect
wrapper Object Basic Css Styles for the tooltip wrapper
content Object Basic Css Styles for the tooltip content area
arrow Object Basic Css Styles for the tooltip arrow container
arrow-inner Object Basic Css Styles for the tooltip arrow

border

Set border width of tooltip-box

Default Value : 1px

Expected Value: 1px, 2px, etc

position

default position of tooltip wrt container

Default Value : n

Expected Value: n|e|w|s|nw|ne|sw|se|g

events - hover

Set trigger for displaying the tooltip

Default Value : true

Expected Value: true|false

events - click

Set trigger for displaying the tooltip

Default Value : false

Expected Value: true|false

events - focus

Set trigger for displaying the tooltip

Default Value : false

Expected Value: true|false

animate - effect - show

Default Value : fade

Expected Value: fade

animate - effect - hide

Default Value : fade

Expected Value: fade

animate - duration

Default Value : 500

Expected Value: Integer

wrapper

Basic Css Styles for the tooltip wrapper

content

Basic Css Styles for the tooltip wrapper

arrow

Basic Css Styles for the tooltip wrapper

arrow-inner

Basic Css Styles for the tooltip wrapper

Browser Support

Chrome, Mozilla firefox
